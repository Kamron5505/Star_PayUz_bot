# 🌟 Star_payuz Telegram Bot

Бот для продажи цифровых товаров и услуг.

## 📦 Установка

1. Установи Python 3.8+
2. Установи зависимости:
```bash
pip install -r requirements.txt
```

## ⚙️ Настройка

1. Открой `config.py`
2. Замени `YOUR_BOT_TOKEN_HERE` на токен от @BotFather
3. Добавь свой ID в `ADMIN_IDS` (узнай у @userinfobot)
4. Укажи свой канал в `CHANNEL_ID` и `CHANNEL_URL`
5. Загрузи баннер и укажи URL в `BANNER_URL`

## 🚀 Запуск

```bash
python bot.py
```

Для админ панели используй команду `/admin` в боте.

## 📋 Функции

✅ Красивое приветствие с баннером
✅ Проверка подписки на канал
✅ Инлайн кнопки услуг
✅ Способы оплаты (Click, Payme, UzCard, Humo)
✅ Админ панель
✅ Уведомления админам о новых заказах
✅ Авто-рассылка
✅ База данных SQLite

## 💡 Команды

- `/start` - Главное меню
- `/help` - Помощь
- `/qoyda` - Связаться с поддержкой
- `/admin` - Админ панель (только для админов)
